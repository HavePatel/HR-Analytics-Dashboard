# Project Overview

This project is a Power BI–based HR Analytics Dashboard designed to analyze workforce metrics and provide insights into employee distribution, attrition patterns, and organizational demographics.
The dashboard turns raw HR data into clear and interactive visuals that support better decision-making for talent management and workforce planning.

# Objective

The main objective of the dashboard is to:

- Understand employee attrition and key contributing factors
- Provide a detailed view of employee demographics
- Help HR teams identify areas requiring attention
- Support data-driven strategies for employee retention and workforce optimization

# Dashboard Features

## 1. KPI Cards

- Total Employee Count
- Attrition Count
- Attrition Rate (%)
- Average Age
- Average Monthly Income
- Average Years at Company

## 2. Interactive Visuals

- Attrition by Gender
- Attrition by Age Group
- Attrition by Job Role
- Attrition by Education Field
- Attrition by Marital Status
- Employee Distribution by Department
- Salary Insights and Work-Life Balance Patterns

## 3. Filters & Drill-through

- Department-level filtering
- Education field filters
- Role-based breakdowns

The dashboard is designed to be clean, interactive, and business-friendly.

# About the Dataset

The dataset used is a standard HR analytics dataset containing information such as:

- Employee demographics (Age, Gender, Education, Marital Status)
- Employment details (Job Role, Department, Years at Company)
- Salary and work satisfaction metrics
- Attrition status

The dataset is suitable for exploring HR trends, attrition behavior, and workforce segmentation.

# Tools Used

- Power BI Desktop – Data modeling & visualization
- DAX (Data Analysis Expressions) – Calculated measures and KPIs
- Power Query – Data cleaning and transformation
- Excel /CSV – Data cleaning and preparation
